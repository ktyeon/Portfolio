C# Console Game

Overview

A console-based game implemented in C# where players navigate through falling objects (enemies) to avoid collisions and accumulate points. This project demonstrates fundamental programming concepts and C# capabilities while creating an engaging gaming experience.

Technical Features

Core Game Development

  • Console application development using C# and .NET framework
  • Implementation of game loop and user input handling
  • Threading control for game speed management

Game Mechanics

  • Random enemy position generation
  • Collision detection system
  • Score tracking and management

Technical Implementation

  • Modular code organization across multiple files
  • File I/O operations for ranking system
  • Console window size and position control
  • Robust exception handling for stability

Key Learning Outcomes

  • Understanding of C# fundamentals and .NET framework
  • Experience with game loop implementation
  • Practice with file operations and data persistence
  • Implementation of basic threading concepts
