    Overview

    A modern implementation of the classic Tetris game using HTML, CSS, and JavaScript. This project demonstrates front-end development skills while creating an engaging and interactive gaming experience with progressive difficulty.

    Game Controls

    ← → Move block left/right
    ↑ Rotate block
    Shift Quick drop

    Technical Features

    Core Game Mechanics

    • Block movement and rotation system
    • Line clearance and scoring mechanism
    • Progressive difficulty with speed increase every 10 points

    Technical Implementation

    • Responsive keyboard controls for user interaction
    • Real-time game state management
    • Dynamic score tracking and display
    • Game over detection and restart functionality

    Development Highlights

    • Clean, modular code architecture
    • Efficient collision detection system
    • Smooth animations and transitions
    • Responsive design for various screen sizes

    Future Enhancements

    • Additional block styles and themes
    • Multiplayer functionality
    • High score leaderboard
    • Sound effects and background music
